Stream from OBS, Omlet & more

Connect OBS Studio, Omlet, or any RTMP app to stream your game, video, or screen live to NOK Super.

OBS Studio

  1. Open OBS Studio → Settings → Stream
  2. Service: Custom...
  3. Server: paste the Server URL above
  4. Stream key: paste the Stream key above
  5. Click Start Streaming. Your game, display, or camera will go live.

Omlet Arcade / Omlet

  1. Open Omlet and start a stream.
  2. In stream settings, choose Custom RTMP or Custom server.
  3. Server URL: paste the Server URL above.
  4. Stream key: paste the Stream key above.
  5. Start streaming your game or screen.

Any RTMP app

  1. In your app’s stream settings, set RTMP URL = Server URL above.
  2. Set Stream key = Stream key above.
  3. Start streaming. Works for game capture, screen share, or video.
